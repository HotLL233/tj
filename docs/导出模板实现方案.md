# 导出模板实现方案

## 模板文件
`D:/桌面/导出模板.xlsx`

## 模板结构总览（10 Sheet）

| # | Sheet 名称 | 维度 | 模板内说明 | 数据类型 |
|---|-----------|------|-----------|---------|
| 1 | 各实验室项目方法对应表 | 明细表 | 无说明 | 树形展开，带 SUM 公式 |
| 2 | 仪器-汇总 | 仪器 | 以仪器为主，按**每日**每台仪器→实验室→项目→方法分类汇总 | 明细列表 |
| 3 | 项目-汇总 | 项目 | 以项目为主，按**每日**每个项目→实验室→仪器→方法汇总，**金额取自方法金额** | 明细列表，带金额公式 |
| 4 | 实验室-汇总 | 实验室 | 以实验室为主，按**每日**每个实验室→项目→仪器→方法汇总，**金额取自方法金额** | 明细列表，带金额公式 |
| 5 | 人员-汇总 | 人员 | **直接提取录入原始记录**（逐条，不做聚合） | 逐条记录列表 |
| 6 | 人员汇总表 | 人员汇总 | 以人员为主，按检测类型分类汇总，**系数相同的归类** | 聚合统计，带公式 |
| 7 | 实验室总表 | 实验室汇总 | 以实验室为主，按项目→检测类型汇总，**金额相同的归类** | 聚合统计，带公式 |
| 8 | 项目总表 | 项目汇总 | 以项目为主，按实验室→检测类型汇总，**金额相同的归类** | 聚合统计，带公式 |
| 9 | 仪器汇总表 | 仪器汇总 | 仪器编号从方法名中括号提取，按类型识别归类 | 聚合统计 |
| 10 | 理化汇总表 | 理化 | 提取理化项下所有相同方法数据加和 | 聚合统计 |

## 每 Sheet 详细实现

### Sheet 1: `各实验室项目方法对应表`

```
列：A=使用实验室, B=项目代号, C=液相仪器, D=检测方法, E=检测数量, F=液相检测量, G=气相检测量, H=项目检测总量
数据来源：project_groups → projects（取 - 前为代号）→ project_method_links → methods
数量列：work_records 按项目/方法聚合 SUM(quantity)，按选定日期范围过滤
公式：
  F = SUM(E{lc_first}:E{lc_last})     -- 液相分组合计
  G = SUM(E{gc_first}:E{gc_last})     -- 气相分组合计
  H = SUM(F{row}, G{row})             -- 项目总量 = 液+气
  总计行 F/G/H = SUM(F{2}:F{n-1})    -- 整列求和
分组：按实验室合并 A 列，按项目代号合并 B 列，按项目合并 F/G/H 列
起始行：数据从第 2 行开始，表头在行 1（0-indexed: HR=0, data_start=1）
所有数据（实验室名/项目名/方法名/仪器编号）从数据库动态读取，不硬编码
```

### Sheet 2: `仪器-汇总`

```
列：A=日期, B=仪器, C=实验室, D=项目, E=方法, F=数量, G=按天数量总计
数据来源：work_records 按 date(recorded_at) + 仪器分组
仪器编号提取：从 methods.full_name 中括号内容提取，如 "[LC-01]"
排序：按日期→仪器→实验室→项目
```

### Sheet 3: `项目-汇总`

```
列：A=项目, B=实验室, C=仪器, D=方法, E=数量, F=方法对应金额, G=金额总计, H=项目金额
金额列 F：= methods.amount（来自方法管理页面设置的金额）
公式：G = E × F, H = SUM(同项目所有 G)
分组：每个项目一个汇总块，合并 H 列
```

### Sheet 4: `实验室-汇总`

```
列：A=实验室, B=项目, C=仪器, D=方法, E=数量, F=方法对应金额, G=数量总计, H=金额总计, I=实验室汇总
金额列 F：= methods.amount
公式：G = E, H = G × F, I = SUM(同实验室所有 H)
分组：每个实验室一个汇总块，合并 A 列和 I 列
```

### Sheet 5: `人员-汇总`

```
列：录入时间, 实验室, 研发项目, 方法, 检测类型, 数量, 录入人
数据来源：work_records 直接 SELECT（带 JOIN 查名），不聚合，逐条列出
检测类型：从 method_type_links → method_types.name 获取
排序：按 recorded_at DESC
```

### Sheet 6: `人员汇总表`

```
列：A=人员, B-D=液相(数量/系数/汇总), E-G=气相(数量/系数/汇总), H-J=理化(数量/系数/汇总), K=个人汇总, L=工作占比, M=总额, N=平均值
聚合规则：GROUP BY user_name, 检测类型, coefficient（系数相同的归一行）
检测类型：从 method_type_links → method_types.name 识别（液相/气相/理化/质谱等）
系数：= methods.coefficient
公式：
  各类型汇总 = 数量 × 系数（如 D=B×C）
  个人汇总 K = 液相汇总 + 气相汇总 + 理化汇总
  工作占比 L = K ÷ N（平均值）
  总额 M = 所有人 K 之和
  平均值 N = AVERAGE(所有人 K)
```

### Sheet 7: `实验室总表`

```
列：A=实验室, B=项目, C-E=液相(数量/金额/汇总), F-H=气相(数量/金额/汇总), I-K=理化(数量/金额/汇总), L=项目汇总, M=实验室汇总
聚合规则：GROUP BY lab, project, 检测类型, amount（金额相同的归一行）
金额：= methods.amount
公式：
  各类型汇总 = 数量 × 金额（如 E=C×D）
  项目汇总 L = 液相汇总 + 气相汇总 + 理化汇总
  实验室汇总 M = 同实验室所有项目汇总之和
```

### Sheet 8: `项目总表`

```
列：A=项目, B-D=液相(数量/金额/汇总), E-G=气相(数量/金额/汇总), H-J=理化(数量/金额/汇总), K=项目汇总
聚合规则：GROUP BY project, 检测类型, amount（金额相同的归一行）
公式：同 Sheet 7
```

### Sheet 9: `仪器汇总表`

```
列：A=仪器编号, B=检测量, C=类型, D=按类型汇总
仪器编号：从 methods.full_name 中括号提取
类型识别：LC-xx → lc, GC-xx → gc, 其他 → icpoes/icpms 等
公式：D = SUM(同类型所有仪器 B 列)
```

### Sheet 10: `理化汇总表`

```
列：A=方法名, B=数量
筛选：WHERE 检测类型='理化'
聚合：GROUP BY method.name
排序：按 SUM(quantity) DESC
无公式，直接填数值
```

## 技术要点

1. **行号**：A-H 列表头在行 1，数据从行 2 开始，公式行号 = 0-indexed + 1
2. **分组合并**：实验室列(A)、项目代号列(B)、汇总公式列(F/G/H) 按分组范围 merge_range
3. **类型识别**：LC-xx → 液相，GC-xx → 气相，理化类型从 method_types 表获取
4. **仪器编号**：从 methods.full_name 中 `[XX-XX]` 格式提取
5. **金额/系数相同归类**：Sheet 6 按 coefficient 分组，Sheet 7/8 按 amount 分组
