# 工作量统计工具 — UI 设计系统 v2.4

---

## 一、设计理念

**清新现代 · 简洁高效** — 以蓝紫渐变为核心色系，毛玻璃 + 微阴影构建层次感，全局 2px 小圆角保持利落。

---

### 2.1 色板

```css
/* 主色 — 蓝紫渐变 */
--primary:        #667eea;
```

```css

/* 悬停提升 */
```

### 2.5 间距 & 断点

### 2.6 动效

```css

  100% { transform: scale(1); opacity: 1; }
}
```

## 三、组件库

### 3.1 布局组件

#### Layout（全局布局）
```
┌─────────────────────────────┐
│  AppBar (毛玻璃, sticky)     │  ← 标题 + Pill导航 + 关于按钮
│  ┌───────────────────────┐  │
│  │     <Outlet />        │  │  ← 页面内容
│  │                       │  │
│  └───────────────────────┘  │
│  BottomNav (移动端固定底部)   │  ← 主页/录入/统计/管理
└─────────────────────────────┘
```

**特性**：
- 毛玻璃顶栏：`backdrop-filter: blur(10px)` + `rgba(255,255,255,0.85)`
- Pill 导航：非激活灰色圆角按钮，激活蓝紫渐变+阴影
- 移动端：底部导航 + 汉堡菜单抽屉
- 关于对话框：动态获取服务端版本号

#### 移动端侧边栏
- 宽度 250px
- 激活项：`bgcolor: rgba(102,126,234,0.12)` + 紫色文字加粗
- 背景色：`#f8fafc`

## 七、MUI Theme 配置

```tsx
// main.tsx
const theme = createTheme({
  palette: {
    primary:   { main: '#667eea' },
    secondary: { main: '#764ba2' },
    background:{ default: '#f8fafc' },
  },
  shape: { borderRadius: 2 },
  typography: {
    fontFamily: '..., "Noto Sans SC", sans-serif',
    h4: { fontWeight: 700 },
    h5: { fontWeight: 700 },
    h6: { fontWeight: 700 },
  },
  components: {
    MuiPaper:  { /* 移除默认背景图, 自定义阴影 */ },
    MuiButton: { /* borderRadius:2, textTransform:none, fontWeight:600 */ },
    MuiChip:   { /* borderRadius:2 */ },
    MuiDialog: { /* borderRadius:2 */ },
  },
  breakpoints: {
    values: { xs:0, sm:640, md:768, lg:1024, xl:1280 },
  },
});
```
