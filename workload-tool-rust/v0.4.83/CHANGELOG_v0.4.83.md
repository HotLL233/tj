# v0.4.83 更新说明

## 本次更新
- 研发送样统计页面复刻分析检测统计页面的卡片、二级统计页、图表面板、导出预览和记录明细交互。
- 研发送样统计数据保持独立：页面调用 `rd-stats`、`rd-records`、`rd-export` 系列接口，后端聚合查询使用 `rd_work_records`，不与分析检测 `work_records` 混用。
- 研发送样统计新增人员、事业部、仪器、项目、类型等聚合数据加载，并补齐 Sheet11 预览入口。
- 研发送样统计接入登录用户作用域：无查看全部统计权限时默认查看本人数据，并限制导出全部统计报表。
- 研发送样记录行内编辑补齐倍率和高项字段更新能力，匹配复刻后的统计明细交互。
