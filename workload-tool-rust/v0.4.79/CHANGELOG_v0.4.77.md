# v0.4.77 版本更新说明

## 本次更新

1. 管理页改为二级页面结构
- `/manage` 仅保留系统管理卡片首页。
- 研发项目、实验室、部门、主数据一键导入、检测方法、回收站、审计日志、数据备份、教程与帮助、样品信息登记、用户管理、页面布局、录入表单、导出模板全部改为独立二级页面。
- 角色管理新增 `/manage/roles` 入口，并保留旧的 `/admin/roles` 兼容地址。

2. 管理二级页面交互统一
- 各二级页面顶部统一增加“返回系统管理”入口。
- 保留原有项目、实验室、部门、检测方法编辑弹窗和保存逻辑，不改动后端业务接口。

3. 部门默认数据重置
- 默认部门调整为：`研究院`、`化工一部`、`化工二部`、`动保一部`、`动保二部`。
- 升级时会执行一次性迁移：旧的激活部门将被停用，实验室与用户上的部门关联将解除，再写入新的 5 个默认部门。
- 历史记录中的部门快照不做回写，避免破坏历史数据关联。

4. 模板与打包版本同步
- 主数据一键导入模板文件名升级为 `v0.4.77`。
- 安装脚本升级为 `样品管理系统_v0.4.77_Setup.exe`，保留同一 `AppId` 和安装目录，支持覆盖安装上个版本。
