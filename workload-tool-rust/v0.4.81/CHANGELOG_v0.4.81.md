# v0.4.81 更新说明

发布日期：2026-07-16

本次迭代在独立目录 `v0.4.81` 内完成。创建新版本目录时未从旧版本复制 `target`、`node_modules`、`dist`、`installer` 等构建产物；构建过程中生成的产物仅保留在 `v0.4.81` 内。

## 本次修改

1. 记录表格列宽优化
- 研发送样记录二级页面：按字段含义重新分配列宽，序号、部门、实验室、项目、类型、数量、操作等短列收窄，方法、备注、日期时间等长列保留换行空间。
- 统计页面检测人记录：新增固定列宽模型，日期时间和方法列自动换行，短字段不再占用过宽空间。
- 分析检测录入页、研发送样录入页的今日记录：统一改为按列宽累计的表格布局，减少短列空占和长方法名截断。

2. 应用图标修复
- 新增 Windows 资源编译脚本，将 `icon.ico` 嵌入 `workload-tool.exe`。
- 安装包开始菜单快捷方式和桌面快捷方式显式指定 `{app}\icon.ico`。
- 解决桌面快捷方式显示为 Windows 默认白页图标的问题。

3. 安装信息与版本
- 应用版本升级为 `0.4.81`。
- 前端版本同步为 `0.4.81`。
- 安装包输出文件名更新为 `样品管理系统_v0.4.81_Setup.exe`。
- 安装包继续沿用相同 `AppId` 和安装目录，可覆盖上一个安装版本。
- 设置卸载显示名为 `样品管理系统`，避免程序列表名称被拼成“样品管理系统 版本 x”的形式。
