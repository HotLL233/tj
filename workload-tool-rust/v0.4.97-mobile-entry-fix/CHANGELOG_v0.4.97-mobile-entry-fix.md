# v0.4.97 手机录入与倍率修复版

## 手机送样录入

- 390px 手机宽度下，送样人/部门、实验室/项目、数量/批号成对两列展示。
- 样品主要成分、备注和附件保持整行，保证长内容输入空间。
- 添加、删除和重置按钮等宽排列，提交登记按钮独占整行。
- 页面和录入区域均不产生横向溢出。

## 检测人记录倍率

- 恢复分析检测统计中“检测人记录”的单价倍率列。
- 单价倍率可直接输入，失焦后调用现有记录更新接口保存。
- 保存失败时恢复原值并显示错误信息。
- 倍率只保存在原始分析检测记录，并继续用于实验室、项目等金额类汇总。
- 人员汇总、人员图表和人员管理分值仍不应用单价倍率。

## 覆盖安装

- 沿用原 AppId、应用名称和安装目录，可覆盖安装 `v0.4.96`。
- 仅替换程序和静态资源，不删除 `{app}\data`。
- 正式程序使用 `cargo build --release`，不启用控制台特性。

## 验证结果

- 前端 TypeScript 与 Vite 构建通过。
- Rust 测试 17 项全部通过，release 构建通过。
- 390px 手机视口无横向溢出，录入字段和操作按钮完整显示。
- 检测人记录倍率修改后可持久化，人员统计分值不应用倍率。
- 正式程序 `/api/health` 返回 `ok`，`/api/version` 返回 `0.4.97`。
- 正式 exe 的 PE 子系统为 Windows GUI，不显示控制台窗口。
- Inno Setup 6.7.2 编译安装包通过，覆盖安装标识与 `v0.4.96` 保持一致。
