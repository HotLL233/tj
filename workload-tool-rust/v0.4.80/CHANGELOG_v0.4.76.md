# v0.4.76 版本更新说明

## 更新日期

2026-07-16

## 新增：主数据一键导入

- 管理页新增“主数据一键导入”入口，可下载系统生成的 Excel 模板。
- 支持批量导入部门、实验室、检测类型、检测方法、研发项目及高项信息。
- 支持在模板中维护项目与实验室、检测方法之间的关联关系。
- 导入前提供预检结果，逐项显示新增、更新、跳过、警告和错误。
- 提供“同名更新”和“同名跳过”两种策略；正式导入使用数据库事务，任一错误会整体回滚。

## 新增：统计图表

- 统计页顶部 6 个汇总卡片新增迷你趋势图或占比图。
- 下方 14 类统计卡片新增图表预览。
- 点击统计卡片后，在原有明细表上方展示对应完整图表。
- 完整图表支持折线图、柱状图和饼图切换，并可切换数量、记录数、管理分值。
- 日期范围和原有筛选条件变化后，卡片与完整图表同步刷新，不改变原统计口径。
- 完成桌面端和移动端响应式适配。

## 升级与数据

- 安装包沿用原 `AppId` 和安装目录，可直接覆盖升级 v0.4.75。
- 升级仅替换程序与前端静态资源，不删除 `{app}\data` 用户数据目录。
- 安装时由 Inno Setup 尝试关闭正在运行的旧程序，安装后不自动重启被关闭的进程。

## 验证

- `cargo test`：3 项导入相关测试全部通过。
- `npm run build`：TypeScript 检查与前端生产构建通过。
- API 实测：模板下载、预检、同名更新、同名跳过、事务导入均通过。
- UI 自动化：6 个顶部迷你图、完整饼图、导入预检、移动端无横向溢出均通过。
- `npm audit`：开发工具链 Vite 存在 1 个高危和 1 个中危告警；官方自动修复要求升级到 Vite 8，属于跨主版本变更，本迭代未强制升级。生产安装包不包含 Vite 开发服务器。
