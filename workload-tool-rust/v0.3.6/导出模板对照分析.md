## 📊 模板结构分析（10 Sheet）

根据"导出模板实现方案.md"，模板包含以下 10 个 Sheet：

### Sheet 1: 各实验室项目方法对应表
**结构**: 树形明细表  
**列**: A-H 列（使用实验室、项目代号、液相仪器、检测方法、检测数量、液相检测量、气相检测量、项目检测总量）  
**特点**:
- 实验室列(A)按分组合并
- 项目代号列(B)按项目合并
- 液相/气相自动识别（LC-/GC-前缀）
- 带 SUM 公式汇总

### Sheet 2: 仪器-汇总
**维度**: 按仪器  
**结构**: 按**每日**每台仪器→实验室→项目→方法分类汇总  
**列**: 日期、仪器、实验室、项目、方法、数量、按天数量总计

### Sheet 3: 项目-汇总
**维度**: 按项目  
**结构**: 按**每日**每个项目→实验室→仪器→方法汇总  
**特点**: **金额取自方法金额**（methods.amount）  
**列**: 项目、实验室、仪器、方法、数量、方法对应金额、金额总计、项目金额

### Sheet 4: 实验室-汇总
**维度**: 按实验室  
**结构**: 按**每日**每个实验室→项目→仪器→方法汇总  
**特点**: **金额取自方法金额**  
**列**: 实验室、项目、仪器、方法、数量、方法对应金额、数量总计、金额总计、实验室汇总

### Sheet 5: 人员-汇总
**维度**: 人员原始记录  
**结构**: **直接提取录入原始记录**（逐条，不做聚合）  
**列**: 录入时间、实验室、研发项目、方法、检测类型、数量、录入人

### Sheet 6: 人员汇总表
**维度**: 按人员聚合  
**结构**: 按人员→按系数分组（液相/气相/理化各有独立系数）  
**列**: 人员、液相数量、液相系数、液相汇总、气相数量、气相系数、气相汇总、理化数量、理化系数、理化汇总、个人汇总、工作占比、总额、平均值

### Sheet 7: 实验室总表
**维度**: 按实验室+金额分组  
**结构**: 按实验室→项目→按金额分组（液相/气相/理化按 amount 分组）  
**列**: 实验室、项目、液相数量、液相金额、液相汇总、气相数量、气相金额、气相汇总、理化数量、理化金额、理化汇总、项目汇总、实验室汇总

### Sheet 8: 项目总表
**维度**: 按项目+金额分组  
**结构**: 按项目→按金额分组（液相/气相/理化按 amount 分组）  
**列**: 项目、液相数量、液相金额、液相汇总、气相数量、气相金额、气相汇总、理化数量、理化金额、理化汇总、项目汇总

### Sheet 9: 仪器汇总表
**维度**: 按仪器类型  
**结构**: 仪器编号→按类型识别归类（LC→液相、GC→气相、ICP→ICP）  
**列**: 仪器编号、检测量、类型、按类型汇总

### Sheet 10: 理化汇总表
**维度**: 理化检测专项  
**结构**: 方法名→数量，仅统计 method_type='理化' 的记录  
**列**: 方法名、数量

fn identify_instrument_type(instrument: &str) -> &str {
    if instrument.starts_with("LC-") || instrument.starts_with("HPLC-") {
        "液相"
    } else if instrument.starts_with("GC-") {
        "气相"
    } else if instrument.starts_with("ICP-") {
        "ICP"
    } else {
        "其他"
    }
}

// 从 method_types 表获取准确类型
SELECT mt.name 
FROM method_type_links mtl 
JOIN method_types mt ON mtl.method_type_id = mt.id 
WHERE mtl.method_id = ?
