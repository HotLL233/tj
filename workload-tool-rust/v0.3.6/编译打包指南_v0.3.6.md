# v0.3.6 版本编译打包指南

## 版本说明

**版本号**: 0.3.6  
**更新日期**: 2026-07-03  
**主要更新**: 导出模块重构，实现 10 个 Sheet 的 Excel 导出

## 更新内容

### 1. 导出模块完全重写

#### 新增 10 个 Sheet：
1. **各实验室项目方法对应表** - 树形结构展示实验室→项目→方法
2. **仪器-汇总** - 按日期、仪器维度汇总
3. **项目-汇总** - 按项目维度汇总，含金额计算
4. **实验室-汇总** - 按实验室维度汇总，含金额计算
5. **人员-汇总** - 原始记录逐条列出
6. **人员汇总表** - 按人员、检测类型、系数聚合
7. **实验室总表** - 按实验室、项目、检测类型、金额聚合
8. **项目总表** - 按项目、检测类型、金额聚合
9. **仪器汇总表** - 仪器编号按类型识别归类
10. **理化汇总表** - 理化检测类型专项汇总

#### 重写的文件：
- `src/api/export_data.rs` - 数据查询层（~700 行）
- `src/api/export_write.rs` - Excel 写入层（~800 行）
- `src/api/export_handler.rs` - 控制层（~170 行）

### 2. 核心功能

- **从方法表获取金额和系数**: 通过 `methods.amount` 和 `methods.coefficient`
- **检测类型识别**: 从 `method_type_links` 和 `method_types` 表关联
- **仪器编号提取**: 从 `methods.full_name` 中提取 `[LC-01]` 格式
- **智能分组聚合**: 按金额相同、系数相同自动归类
- **公式化汇总**: 液相/气相/理化分类汇总，项目/实验室/人员多级汇总
- **单元格合并**: 实验室列、项目列、汇总列智能合并

### 使用 PowerShell 脚本自动打包

创建文件 `pack.ps1`:

```powershell
# v0.3.6 自动打包脚本
$version = "0.3.6"
$projectRoot = "D:\桌面\工作量统计工具项目\workload-tool-rust\v0.3.5"
$packageRoot = "D:\桌面\工作量统计工具项目\workload-tool-rust\安装包"
$packageDir = "$packageRoot\v$version"

Write-Host "开始打包 v$version..." -ForegroundColor Green

# 创建打包目录
New-Item -ItemType Directory -Force -Path $packageDir | Out-Null

# 复制可执行文件
Copy-Item "$projectRoot\target\release\workload-tool.exe" -Destination $packageDir -Force
Write-Host "  [√] 复制可执行文件" -ForegroundColor Green

# 复制静态资源
if (Test-Path "$projectRoot\static") {
    Copy-Item "$projectRoot\static" -Destination "$packageDir\static" -Recurse -Force
    Write-Host "  [√] 复制静态资源" -ForegroundColor Green
}

# 复制配置文件示例
if (Test-Path "$projectRoot\config.toml") {
    Copy-Item "$projectRoot\config.toml" -Destination "$packageDir\config.toml.example" -Force
    Write-Host "  [√] 复制配置文件" -ForegroundColor Green
}

# 复制图标
if (Test-Path "$projectRoot\icon.ico") {
    Copy-Item "$projectRoot\icon.ico" -Destination $packageDir -Force
    Write-Host "  [√] 复制图标" -ForegroundColor Green
}

# 复制 README
if (Test-Path "$projectRoot\README.md") {
    Copy-Item "$projectRoot\README.md" -Destination $packageDir -Force
    Write-Host "  [√] 复制 README" -ForegroundColor Green
}

Write-Host "`n打包完成！" -ForegroundColor Green
Write-Host "安装包位置: $packageDir" -ForegroundColor Cyan
Write-Host "`n文件列表:" -ForegroundColor Yellow
Get-ChildItem $packageDir -Recurse | Select-Object FullName
```

运行打包脚本：

```powershell
cd D:\桌面\工作量统计工具项目\workload-tool-rust\v0.3.5
.\pack.ps1
```
