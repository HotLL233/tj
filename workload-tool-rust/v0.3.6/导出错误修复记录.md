## ✅ 已修复的问题

### 1. **日期处理错误** (export_handler.rs)

**问题**: 使用了 `Box::leak` 导致内存泄漏和潜在的生命周期问题

**修复前**:
```rust
(Box::leak(month_start.into_boxed_str()) as &str,
 Box::leak(end_date.into_boxed_str()) as &str)
```

**修复后**:
```rust
let start_str: String;
let end_str: String;
let (start, end) = if let Some(ref s) = q.start {
    start_str = s.clone();
    end_str = q.end.as_ref().cloned().unwrap_or_else(|| s.clone());
    (start_str.as_str(), end_str.as_str())
} else {
    // 正确的日期计算逻辑
    let now = chrono::Local::now();
    start_str = format!("{}-{:02}-01", now.year(), now.month());
    
    let last_day = if now.month() == 12 {
        chrono::NaiveDate::from_ymd_opt(now.year() + 1, 1, 1)
            .and_then(|d| d.pred_opt())
    } else {
        chrono::NaiveDate::from_ymd_opt(now.year(), now.month() + 1, 1)
            .and_then(|d| d.pred_opt())
    };
    
    end_str = if let Some(d) = last_day {
        d.format("%Y-%m-%d").to_string()
    } else {
        format!("{}-{:02}-28", now.year(), now.month())
    };
    
    (start_str.as_str(), end_str.as_str())
};
```

### 2. **Sheet 6 公式引用错误** (export_write.rs)

**问题**: 工作占比公式引用了错误的单元格

**修复前**:
```rust
// 引用 CN 列的 HR+2 行（第2行），但平均值在 HR+1 行（第1行）
ws.write_formula(r, CL, format!("={}{}/{}{}", 
    col_letter(CK), r+1, col_letter(CN), HR+2))?;
```

**修复后**:
```rust
// 正确引用总额列 CM 的 HR+2 行
ws.write_formula(r, CL, format!("={}{}/{}{}", 
    col_letter(CK), r+1, col_letter(CM), HR+2))?;
```

### 3. **增强错误处理** (export_handler.rs)

**问题**: 错误时不知道哪个 Sheet 出问题

**修复**: 为每个 Sheet 添加详细的错误捕获和日志

```rust
// ========== Sheet 1: 各实验室项目方法对应表 ==========
match (|| -> Result<()> {
    let data = export_data::query_sheet1_data(&conn, start, end, q.group_id)?;
    tracing::info!("Sheet 1 查询完成: {} 行", data.len());
    let ws = wb.add_worksheet();
    export_write::write_sheet1(ws, &data, &fmt)?;
    tracing::info!("Sheet 1 写入完成");
    Ok(())
})() {
    Ok(_) => {},
    Err(e) => {
        tracing::error!("Sheet 1 失败: {}", e);
        return Err(e);
    }
}
```

现在每个 Sheet 的查询和写入都会分别记录日志，失败时能精确定位。

### 4. **移除不必要的 .as_str() 调用** (export_write.rs)

**问题**: 在公式字符串上多余的 `.as_str()` 调用

**修复**: 批量移除，`format!()` 返回的 `String` 可以直接传递给 `write_formula()`
