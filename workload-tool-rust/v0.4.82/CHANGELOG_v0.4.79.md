# v0.4.79 更新说明

发布日期：2026-07-16

本次迭代在独立目录 `v0.4.79` 内完成，未改动历史版本目录代码。

## 本次修改

1. 管理与统计权限
- 新增统计管理权限组，纳入角色权限矩阵。
- 为统计页面补充卡片级权限点：
  `stats:workload:week`、
  `stats:workload:month`、
  `stats:workload:user-log`、
  `stats:workload:division`、
  `stats:workload:sheet1` ~ `stats:workload:sheet10`、
  `stats:workload:view-all`。
- 默认角色“分析检测员”补齐统计页访问及卡片查看权限。

2. 统计页访问范围控制
- 无“查看全部统计数据”权限的用户，进入统计页后默认只看本人数据。
- 统计卡片按权限显示，未授权卡片不展示。
- 检测人记录筛选默认锁定为当前登录用户。
- 无“查看全部统计数据”权限时，禁用 Excel 导出入口，避免导出超范围数据。

3. 统计明细显示优化
- 检测人记录中的“日期时间”列支持自动换行。
- 检测人记录中的“方法”列支持自动换行，避免长方法名被截断。

4. 安装包与版本信息
- 应用版本升级为 `0.4.79`。
- 前端版本同步为 `0.4.79`。
- 安装脚本输出文件名更新为 `样品管理系统_v0.4.79_Setup.exe`。
- 安装包沿用相同 `AppId` 和安装目录，可覆盖上一个安装版本。
