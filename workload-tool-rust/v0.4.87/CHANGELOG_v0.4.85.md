# v0.4.85 更新说明

## 研发送样记录

- 新增“送样时间”列，显示研发送样记录的 `recorded_at`，与“取样时间”分开展示。
- 记录编辑改为行内编辑：点击编辑后，当前记录行的字段直接切换为输入框/下拉框，不再在记录下方展开编辑区域。
- 支持在行内编辑送样人、部门、实验室、项目、方法、数量、批号、送样时间和注意事项。
- 日期时间列支持日期、时间分行显示，长方法、注意事项自动换行。

## 研发送样录入

- 顶部布局改为紧凑工具区：标题、实验室信息、状态、送样时间和提交按钮集中展示。
- 录入表格列宽从百分比压缩改为固定最小宽度，支持横向滚动。
- 增大行内输入框高度和内边距，改善填写时输入框过小的问题。
- 今日记录区同步新增“送样时间”列，并优化长方法、日期时间的显示。

## 兼容与打包

- 后端更新接口补充 `group_id`、`division_id` 保存支持，确保研发送样记录行内编辑实验室、部门后可以正确落库。
- 研发送样列配置内置“送样时间”字段，旧数据库启动后会自动补齐。
- 安装包版本更新为 `0.4.85`，沿用同一 `AppId` 和安装目录，可覆盖上一个版本安装。
