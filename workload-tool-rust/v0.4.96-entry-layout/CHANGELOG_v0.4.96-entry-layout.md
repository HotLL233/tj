# v0.4.96 登记与记录布局优化版

## 布局调整

- 样品信息登记页工作区扩展至 1536px，减少宽屏两侧空白。
- 登记字段改为响应式网格：宽屏多列、中屏四列或两列、手机单列。
- 每条待登记样品使用独立紧凑区域，保留多行添加、选择删除、重置和统一提交。
- 登记记录在 PC 端合并为 9 个信息列，相关字段同格展示并完整换行，不产生页面级横向滚动。
- 中小屏记录自动切换为卡片视图，状态、人员、项目、样品、时间、附件和操作完整展示。
- 编辑功能直接显示在当前记录下方的编辑网格，不再使用点击行折叠详情。
- 样品信息统计页工作区同步扩展至 1536px。

## 逻辑保持

- 未修改样品登记、查询、状态流转、权限和附件后端接口。
- 保留待取样、待检测、已检测三种状态及原有颜色。
- 保留附件上传、查看和删除能力。
- 保留状态筛选和分页逻辑。

## 页面检查

- 已扫描全部路由页面的固定宽度、表格宽度、横向滚动和移动端断点。
- 研发送样记录、分析检测记录、两套统计页和管理页在 PC/手机均无页面级横向溢出。
- 首页、功能门户和帮助页继续使用阅读型限宽布局，未强制拉伸。

## 覆盖安装

- 沿用旧版 AppId、应用名称和默认安装目录。
- 使用 `UsePreviousAppDir=yes` 复用旧安装目录。
- 升级只替换程序和静态资源，不删除 `{app}\data`。
- 正式程序使用 `cargo build --release` 构建，不启用控制台特性。
