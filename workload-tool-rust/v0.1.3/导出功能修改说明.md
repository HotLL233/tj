### 2. 实验室编号提取
**文件**: `src\api\export_handler.rs` (第58-65行)

**新增函数**:
```rust
fn extract_lab_number(lab_name: &str) -> String {
    // Extract number from lab name like "410实验室" -> "410"
    lab_name.chars()
        .take_while(|c| c.is_numeric())
        .collect::<String>()
        .into()
}
```

**说明**: 从实验室名称（如"410实验室"）中提取编号部分（"410"），用于显示在 Excel 的"使用实验室"列。

### 3. 数据结构调整
**文件**: `src\api\export_handler.rs` (第217行)

**修改前**:
```rust
struct RangeInfo { start: u32, end: u32, lab: String, code: String, pc_rows: u32, lc_rows: u32, gc_rows: u32 }
```

**修改后**:
```rust
struct RangeInfo { start: u32, end: u32, lab: String, lab_number: String, code: String, pc_rows: u32, lc_rows: u32, gc_rows: u32 }
```

**说明**: 
- `lab`: 保留完整的实验室名称（如"410实验室"），用于从数据中查找
- `lab_number`: 新增实验室编号字段（如"410"），用于在 Excel 中显示

### 4. 数据写入调整
**文件**: `src\api\export_handler.rs` (第233行 和 第263行)

**修改**:
- 在创建 `RangeInfo` 时，添加 `lab_number: extract_lab_number(lab_name)`
- 在写入 Excel 时，使用 `rg.lab_number` 而不是 `rg.lab` 来写入实验室列

**说明**: 确保在 Excel 中显示的是实验室编号（如"410"），而不是完整名称（如"410实验室"）。
